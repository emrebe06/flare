# Güvenli Kapsam

Bu uygulama sadece pasif analiz yapar.

Yapılmayanlar:
- IP/konum tespiti
- Kişisel veri ifşası
- Yetkisiz erişim
- Exploit/injection
- Form gönderimiyle test
- Hesap kilitleme
- Karşı tarafa müdahale

Gelecek özellik fikirleri:
- Şikayet PDF üretimi
- Ekran görüntüsü ekleme
- Sahibinden/Dolap/Letgo özel kurallar
- Global marketplace profilleri
- Yerel veritabanı ile ihbar edilen IBAN/telefon hash eşleşmesi
- Reverse image search entegrasyonu sadece resmi API varsa
